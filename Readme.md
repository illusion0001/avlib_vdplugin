# Usage

Copy 'avlib-1.vdplugin' to 'VirtualDub2\plugins64' folder.

Copy FFMpeg DLLs to 'VirtualDub2' folder.

avcodec-60.dll
avformat-60.dll
avutil-58.dll
swresample-4.dll
swscale-7.dll 

FFMpeg 6.1 binaries used: https://www.gyan.dev/ffmpeg/builds/packages/ffmpeg-6.1-full_build-shared.7z
