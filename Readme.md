# Usage

Copy 'avlib-1.vdplugin' to 'VirtualDub2\plugins64' folder.

Copy FFMpeg DLLs to 'VirtualDub2' folder.
* avcodec-61.dll
* avformat-61.dll
* avutil-59.dll
* swresample-5.dll
* swscale-8.dll 

FFMpeg 7.0 binaries used: https://www.gyan.dev/ffmpeg/builds/packages/ffmpeg-7.0-full_build-shared.7z
